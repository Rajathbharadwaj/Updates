In this post, we will cover how to train and deploy a machine learning model leveraging a scalable stream processing architecture for an automated text prediction use-case. Namely, the deployed models can scale to a variable number of replicas, together with autoscaling based on Kubernetes HPA that would horizontally scale based on resource usage. This dataset consists of over 200,000 reddit comments — primarily labelled based on whether the comments have been removed by moderators. Similarly, Kafka can also scale horizontally through the number of brokers, which enables for large throughputs with low latency. The most common approach is using one of the existing prepackaged-model servers. Make learning your daily ritual.
