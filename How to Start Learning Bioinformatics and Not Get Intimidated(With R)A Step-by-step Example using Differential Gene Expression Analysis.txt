We performed our first differential gene expression analysis, and we identified some candidate genes that are upregulated/downregulated when the TCF21 gene is silenced in coronary artery muscle cells! Before we get started, it is important to note that I’ve chosen to use DESeq2 for this analysis, although there exist other packages that do DGEA using different methods. This is the accompanying code to produce a heatmap of distances.Another way to assess similarity between samples is to use a principal component analysis (PCA) plot.
