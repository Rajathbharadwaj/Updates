Consider the following data of four various functions — a linear function, a logarithmic function, an exponential function, and a power function. No matter how many axes we’ve created, our function will always add each line to our legend. Even better, we can easily control which lines are added to the legend by whether or not we supply the label argument when we plot the line. However, if we want to change ordering of the handles in our legend, we could manually pass the axes. Unfortunately, we have to call this for each axes, which means we’d still need to write a for loop and append the handle and label to a list. matplotlib does have a function to obtain the handles and labels! The function returns a dictionary we can then use with the splat operator (**) to unpack as keyword arguments to plt.legend().
