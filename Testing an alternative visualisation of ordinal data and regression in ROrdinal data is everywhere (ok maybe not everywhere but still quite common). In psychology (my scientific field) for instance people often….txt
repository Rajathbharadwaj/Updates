As I mentioned above, plotting conditional effects for ordinal models normally requires to use a dimension for the probability of each response. Let’s consider that the model is ok. Now we need to get estimates from the model because we want to be able to plot it. The dashed lines are simply connecting a kind of “overall” response estimate (computed from the brms model) in each condition. Here I choose to plot this probability (alongside its credibility interval, CI) on another small axis above the jittered dots.
