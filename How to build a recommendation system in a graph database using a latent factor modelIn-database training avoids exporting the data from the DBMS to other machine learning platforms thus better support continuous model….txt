Follow the steps below to build a recommendation with the movie rating records.Run the splitData queryThis query split rating data into the validation set and training set. The movies are recommended based on the rating prediction.Training machine learning model in a graph database has the potential to achieve real-time updates of the recommendation model. The average rating of each movie is computed from the training data.Run the Initialization queryThis query initializes the latent factor vectors for the users and the movies. The query output is all the ratings given by the user and the rating predictionRun the recommend queryThis query output the top-10 movies recommended to a user. Note that the ratings from users to movies from a graph structure: users and movies are the vertices, and ratings are the edges of the graph. Typically, the system provides the recommendation to the users based on its prediction of the rating a user would give to an item.
