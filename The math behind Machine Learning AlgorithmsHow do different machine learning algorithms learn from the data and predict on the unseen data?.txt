The algorithm finds the k nearest neighbors of data points by computing its distance from all the data points. After the assignment of the data points, the centroid of each cluster is computed and again the data points are assigned to the nearest clusters. The data point is assigned to the class with the highest number of points among the k neighbors (voting process). In random forests, each decision tree predicts a class outcome and the class outcome with the most number of votes becomes the prediction of random forests. The clusters formed should be such that the data points within a cluster are as similar as possible and the clusters are as distinct as possible amongst each other. This process is repeated until the data points remain in the same cluster or the centroids of the clusters do not vary each consecutive iterations. This prevents individual trees to predict using the same features.This algorithm can also be used for both, Regression as well as Classification.
