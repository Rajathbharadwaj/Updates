However, different region proposals have different sizes, which also result in different output feature map sizes if we are using the same CNN feature extractor. It is composed of a ResNet backbone, an FPN detection neck to channel features at different scales, and two subnets for classification and box regression as detection head. Feature Pyramid Networks is an attempt to improve the detection head by using features from different layers to form a feature pyramid. Also, we only need 3 anchors with different aspect ratios per location now because objects with different sizes will be handle by different levels of the feature pyramid. Cascade R-CNN’s solution is to chain multiple detection head together, each will rely on the bounding box proposals from the previous detection head. At the same time, Anchor-free detector like CenterNet showed us a promising future where object detection networks can become as simple as image classification networks. In other words, it treats object detection as a regression problem, instead of a classification problem relying on region proposals.
