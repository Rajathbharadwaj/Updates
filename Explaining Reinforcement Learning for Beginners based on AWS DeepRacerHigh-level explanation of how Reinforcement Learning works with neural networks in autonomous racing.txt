It will increase the probability of actions that gave a higher reward while decreasing the probability of actions that led to poor rewards. This algorithm combines two neural networks to learn which actions to take: the policy network and the value network. For some optimization problems, you can arrive at a global set of actions, which maximizes the amount of reward the agent can receive within a given set of constraints. Then this network, based on past experiences, estimates what is the expected reward from taking each of these actions. Roughly speaking, the agent, guided by the algorithm, repeatedly tries different sets of actions, attempting to maximize the total reward received. It will use the expected reward for taking each action and transform them into probabilities, applying a method called policy gradients.
