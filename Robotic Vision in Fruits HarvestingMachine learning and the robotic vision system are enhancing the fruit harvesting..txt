Many algorithms like LaB, color histogram, HOG, watershed, MSRM, CNNs are used for detection, segmentation, counting, yield prediction and, harvesting the fruits in the field. By applying robotic vision, various tasks can be performed into different image regions to identify and segment fruits based on shape, color, texture and size levels. In an unstructured environment researchers are focusing on these problems to make more advance robotic vision systems.This article presents the techniques from basic feature extraction methods to high-level methods. Machine learning and the robotic vision system combination opens a gateway for precision agriculture and enhances the quality of fruit harvesting an application. Many approaches use images that contain fruits to make more an advanced vision system for yield estimation analysis. For this researcher emphasizes the various image processing techniques to solve the hardship of labor in agriculture. The study shows how machine learning techniques join with image processing to make a powerful system.
