Two common tools to improve gradient descent are the sum of gradient (first moment) and the sum of the gradient squared (second momentum). Vanilla gradient descent just follows the gradient (scaled by learning rate). If the decay rate is 0, then it is exactly the same as (vanilla) gradient descent. The following methods do some additional processing of the gradients to be faster and better.The gradient descent with momentum algorithm (or Momentum for short) borrows the idea from physics. In the context of machine learning, the goal of gradient descent is usually to minimize the loss function for a machine learning problem. The average gradient for sparse features is usually small so such features get trained at a much slower rate. This is because the sum of gradient squared only grows and never shrinks.
