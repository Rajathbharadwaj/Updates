Even though the title says the quality of “machine learning software”, principles and ideas can be reused for inspecting any software quality.Let’s consider a Python machine learning application. TensorFlow is in that case a direct dependency of the application and by installing it, the machine learning application is using directly TensorFlow and indirectly dependencies of TensorFlow. Dependency Monkey can in fact create the state space of all the possible software stacks that can be resolved respecting version range specifications. requirements on API offered by tensorflow>=2.0.0), we can test our application with different versions of TensorFlow up to the current 2.3.0 release available on PyPI to this date. This article will cover “inspections” of software stacks and will link a free dataset available on Kaggle. Doing so enables Dependency Monkey to resolve software stacks at a fast pace (see a YouTube video on optimizing Thoth’s resolver). A notebook published together with the dataset can help one exploring the dataset.Project Thoth is an application that aims to help Python developers.
