The class method takes the class as a parameter to know about the state of that class. When we create methods within a class they receive instance as the first argument automatically. These methods are not object-specific.We use class methods with the help of a decorator (“@classmethod”). Imagine the pain if you manually did it for every instance.The methods defined above(new_price, discount_price) are regular methods inside the Mobile Class. The __init__ method automatically gets called whenever you create a new instance of a class. A static method also belongs to the whole class rather than a particular object. Regular methods take instance aka self as the first argument and are used to alter the functionality of an object/instance.
