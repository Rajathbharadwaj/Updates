For any given state, an agent can query a value function to determine the “value” associated with being in that state. Instead, the agent can query its state-action value function for the value of taking the “left” action in state A, q(A, left). It does not need to know that taking the “left” action in state A will lead to state B, which has value v(B). In the state value case, we consider the average reward where the average is taken over all actions according to policy π. Simply put, a policy takes a state and returns a weighting over which actions the agent should take in that state.
