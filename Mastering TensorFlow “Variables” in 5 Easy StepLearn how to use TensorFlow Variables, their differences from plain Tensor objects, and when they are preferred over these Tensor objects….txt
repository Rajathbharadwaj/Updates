Now let’s take a look at the characteristics and features of variables.Every Variable must have some properties such as value, name, uniform data type, shape, rank, size, and more. See the example below:Just as in Tensor objects, you can reshape Variable objects as well. To be able to view a Variable’s values, we can use the .value() function as well as the .numpy() function. Since Variable objects are built on top of Tensor objects, they have common attributes such as .shape and .dtype. Since there is a single type of data stored for every Variable, you can also view this type with the .dtype attribute. Being able to assign new values is one of the advantages of Variables, where value reassignment is required.
