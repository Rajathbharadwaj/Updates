It is very important to test synthetically-trained models on real data to know whether the model is successfully generalizing to real data. We used the technique of domain randomization to improve the performance of our model on real images. There is a gap between simulation-produced data and real data known as the sim2real gap. If you have any questions or are curious about synthetic data, send us an email at info@zumolabs.ai, we love to chat.References Zumo Labs Data Portal. Important to note that if we were using real images for training we would have to manually annotate thousands of images instead of just a handful for testing! As we predicted, the performance of the models increases with the more synthetic data we use. This has the downstream effect of making the model we train on this data more robust to variations in color and lighting.
