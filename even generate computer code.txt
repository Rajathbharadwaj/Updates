At the software giant’s Build conference in May, Sam Altman, a cofounder of OpenAI, demonstrated how GPT-3 could auto-complete code for a developer. There is no public demo of the technology yet, but Bektes, the founder, says for simple commands it works 80 to 90 percent of the time. He seems confident it could change some aspects of software development.
