For loops are an essential aspect of Python code, as well as code in many other languages. For loops are what allow us to iterate over collections like lists, tuples, sets, and dictionaries in Python. By default, if we iterate over a dictionary using just one variable name, only the key will be selected. This method goes through each item in a collection one-by-one and performs whatever operation proceeds the for loop declaration.
