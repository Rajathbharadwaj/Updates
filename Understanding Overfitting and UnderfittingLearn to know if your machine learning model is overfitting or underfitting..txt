But the general idea is when we try to fit a complex model with significantly insufficient training data, then the model tends to overfit the training data. This phenomenon of doing well on test data is known as generalize on test data in machine learning terms. However, as the value of K increases (K = 3, 11) the classifier’s accuracy decreases on training data but increases on test data. Well, model underfitting is the complete opposite of that.When a model underfits, it is likely to do very bad on the training data. So the better a model generalizes on test data, the better the model is. A machine learning model’s true effectiveness actually depends on how well it does on the test set data.
