In your Google Cloud Platform user console, navigate to Dataproc and click on “CREATE CLUSTER”.You’ll be taken to “Create a cluster” page with settings for your cluster of machines. This will allow your cluster to access other Google Cloud services such as BigQuery.In Optional components, we see a variety of components available. Check “Enable access to the web interfaces” under “Component gateway” and expand “Advanced options”.Although optional, set a cloud storage staging bucket. Once the spinning icon next to your cluster name turns to green dot check mark, click on the “web interfaces” tab and “Jupyter” under Component gateway. If Jupyter is not available, make sure you’ve followed the instructions in the previous section to “enable access to web interfaces” and selected the optional components. The next image shows what options are available.Lastly, check “Allow API access to all Google Cloud services” under Project access.
