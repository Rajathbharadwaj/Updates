Relationship is linear — When carrying out linear regression (simple or multiple), the relationship between the independent variable(s) and the dependent variable must be linear. If this is the case, typically you want to include only one of the variables within the pair into the regression model. There is no multicollinearity* amongst the independent variables — Multicollinearity occurs when the independent variables are related to each other. Variables in the dataset come from normally distributed populations — all variables in the data should also arise from normally distributed populations. In each subsequent round, the model tries to improve on model accuracy (accuracy metric selected by user) based on what it has learnt in previous runs. A coefficient term (β) is added for each independent variable (x) included in multiple linear regression.Let’s start with the y-intercept or constant, β₀. It is the value predicted for y if all the independent variables (X) were zero.
