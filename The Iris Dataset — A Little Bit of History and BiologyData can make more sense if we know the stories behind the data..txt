This dataset consists of 50 records for each of three Iris species: Iris setosa, Iris virginica, and Iris versicolor.Beyond that, do you know where the dataset came from? Let’s have a brief getaway from dealing with data and learn some interesting stories behind this common dataset.The Iris flower dataset is also known as the Fisher’s Iris dataset. However, while I’m learning more about data science research, I find it so critical to have an in-depth understanding of the data. What most people know about the Iris dataset is that it has records on the length and width measures for sepals and petals. The first time that I got to know the Iris dataset was when I was learning discriminant analysis using SAS. In this article, Fisher developed and evaluated a linear function to differentiate Iris species based on the morphology of their flowers.
