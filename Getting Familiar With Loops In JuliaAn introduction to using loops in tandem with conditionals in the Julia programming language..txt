In the example of an array, we can loop through each element continuously one by one using an iterative loop. To create a while loop, we will, of course, use the key-word “ while,” followed by a condition.Another thing to look at is the ability to break a loop. A great way to think about recursive looping is to imagine that the function itself is both the loop and the initialization of the loop. Most recursive algorithms are going to use a condition to determine as to whether or not a loop should be broken with a return. Iterative loops are different from recursive and while loops in that they need dims to loop over. The first thing we are going to do in our factorial function is check if the number we want the factorial of is 1. If this were to be the case, the factorial of 1 is of course just one, so we will return 1.This will break our loop.
