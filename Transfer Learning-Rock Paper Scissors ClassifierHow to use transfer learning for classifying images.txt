Building an algorithm is always beneficial but time consuming so why not use existing algorithms and model for similar type of data. As the existing models have few constraints about the size of the input image we need to reshape our data set. The process of using the stored knowledge which is gained while solving one problem and applying it to a different but similar problem is called Transfer Learning. MobileNets come in various sizes controlled by a multiplier for the depth (number of features), and trained for various sizes of input images. It is not trained for our specific task: classifying an image of a hand as rock, paper, or scissors. Growing up building things using Lego has always been fun, so is building machine learning algorithm from scratch. As a result, we get an abrupt classification which is nowhere close to our desired result.
